# Processed Data

This folder contains cleaned and structured datasets prepared for analysis in the USD/CAD exchange rate project.

These datasets were created after collecting raw macroeconomic data from various public sources and performing data preprocessing tasks such as:

- Data cleaning
- Handling missing values
- Aligning datasets by date
- Creating derived variables
- Combining multiple datasets for analysis


The datasets in this folder are used for Power BI data modeling and predictive analysis in Python.
## Files Included

FX_Master.csv  
Combined dataset containing all major macroeconomic indicators used for predictive modeling of the USD/CAD exchange rate.

CPI.csv  
Cleaned inflation data used in the analysis.

FX_Rate.csv  
Historical USD/CAD exchange rate data.

GDP.csv  
Processed GDP data used to represent economic activity.

Gold_Rate.csv  
Historical gold price data used as a commodity indicator.

Oil_Rates.csv  
Crude oil price data, which is an important factor influencing the Canadian dollar.

Policy_Rate_Difference.csv  
Difference between U.S. and Canadian policy interest rates used as a monetary policy indicator.

Trade_Balance.csv  
Trade balance data representing economic trade relationships between Canada and the United States.
